# crowd-sentinel-ai
Real time crowd density detection and alert system using YOLOv8,optimized for fast alert system for intelligent monitoring
🛡️ Crowd Sentinel AI

AI system to detect crowd density in images and videos using YOLOv8.

---

🚀 What it does

- Detects people
- Counts crowd
- Gives alert if crowd exceeds limit
